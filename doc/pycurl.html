PycURL -- A Python interface to the cURL library


The pycurl package is a Python interface to the curl library (http://curl.haxx.se/libcurl/).  pycurl works with Python versions from 1.5.2 and upwards.

The curl library s a client-side URL transfer library supporting FTP, FTPS, HTTP, HTTPS, GOPHER, TELNET, DICT, FILE and LDAP.  The cURL library also supports HTTPS certificates, HTTP POST, HTTP PUT, FTP uploads, proxies, cookies, basic authentication, file transfer resume of FTP sessions, HTTP proxy tunneling and more.

All the functionality provided by the curl library can through the pycurl interface.  The following sections describe how to use the pycurl interface, with emphasis on how the Python interface works instead of how the curl library itself works.  For information on how the curl library works, please consult the curl library web pages (http://curl.haxx.se/libcurl/using/).
